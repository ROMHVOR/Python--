Graph Navigator
Автор: Хворостов Роман

📝 Краткое описание
Graph Navigator - это консольное приложение для работы с графами, реализованное на Python с использованием объектно-ориентированного программирования. Приложение позволяет создавать различные типы графов, выполнять обходы (BFS/DFS), находить кратчайшие пути, а также сохранять и загружать графы в формате JSON.

Основные возможности:
✅ Создание ориентированных, неориентированных и взвешенных графов

✅ Добавление/удаление вершин и рёбер

✅ Обход графа в ширину (BFS) и глубину (DFS)

✅ Поиск кратчайшего пути (алгоритм Дейкстры для взвешенных графов, BFS для невзвешенных)

✅ Сохранение и загрузка графов в формате JSON

✅ Валидация входных данных

✅ Паттерн "Фабрика" для создания различных типов графов

✅ Полная инкапсуляция, наследование и полиморфизм

🚀 Как запустить приложение
Требования
Python 3.8 или выше

Операционная система: Windows/Linux/MacOS
